Melodyne使用

发表于 2019-12-05 | 分类于 效果器 | 阅读次数:
字数统计: 958 | 阅读时长 ≈ 3

Melodyne是强大的，想要学会使用它。

本文忽略Melodyne作为独立软件的一些特有功能，这里只聚焦作为插件时Melodyne的实用操作。

检测算法

不同的算法应对不同的音频内容；

The Melodic algorithm

这是旋律算法，在该算法下，应该尽量使用干净无混响的录音；

Blobs的产生取决于断音和连奏；

The Percussive algorithm

该算法主要应对打击乐器以及噪音等。它们都以相同的音高显示，调整Blob只会以半音显示相对值；

The Polyphonic (Sustain/Decay) algorithm

这是处理复音的算法，可以检测和编辑钢琴或吉他之类的声音，这里存在两个版本：

  • 和弦延音算法，早期版本的Melodyne用户已经很熟悉，并且适用于多种和弦音频材料；
  • 和弦衰减算法是一种变体，特别适合于高频打击乐信号，通过其可以看出音调；

该算法对一些质量的音频存在无法使用的情况；

The Universal algorithm

通用算法特别适合包含打击乐和音调元素的复杂信号。例如，如果您希望更改整个音乐的音调，时间或节奏，则此算法将提供最佳的音质；

Playback, navigation, zooming

双击时间标尺将从双击的地方开始播放，单击时间标尺指针将暂停播放回转到单击处；

在音符编辑器中双击进行播放，再次双击则暂停，指针停留在正播放处；

单击时间标尺并向上向下拖动可以对显示内容进行缩放；

Scrolling and zooming in the Note Editor(音符编辑器的滚动与缩放)

通过手形图标或按下快捷键Command + Shift单击鼠标左键并移动来滚动显示区域；

按住Command + Alt单击鼠标左键并移动来缩放显示区域；

选中一个blob或者多个blobs，按住快捷键Command + Shift，对准选中的内容双击鼠标左键，将对该内容进行针对性的放大，要想还原到之前的缩放效果，则按住快捷键Command + Shift，在空白处双击鼠标左键；

只通过触控板也可以对显示区域进行滚动和缩放(这种方法实际操作体验上略显卡顿)：

  • 用两只手指在触控板上上下左右滑动，可以对显示内容进行上下左右滚动；
  • 在触控板上使用缩放操作，可以对应显示区域的缩放；

Cycle mode(循环模式)

作为插件的时候，Melodyne的循环模式在宿主播放暂停时才有用；

定义循环范围

时间标尺下方有一个横栏，存在一个指向右和左的小角标，叫做循环定位器，通过它来确定循环范围；

使用Alt可以让循环定位器在调整循环范围时不受网格线的影响；

打开或关闭循环模式

可以通过双击时间标尺正下方的窄条中的循环范围来打开或关闭循环模式。 激活循环模式后，循环范围会高亮；

将鼠标指针悬停在循环块中间，按住鼠标左键左右拖动，可以移动循环块，此时如果按住Alt可以忽略时间网格线；

按住Shift，鼠标单击的地方将被包含在循环块中，包含的位置受时间网格线制约，此时如果同时按住Alt将不受时间网格线影响；
