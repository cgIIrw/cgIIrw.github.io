Sylenth1手册简明翻译

发表于 2019-08-29 | 分类于 合成器 | 阅读次数:
字数统计: 6,760 | 阅读时长 ≈ 24

本文是Sylenth1Manual的粗翻。

3. 概览

Sylenth1的用户界面分成四个部分，每个部分拥有不同的颜色以便彼此区分。

第一个部分被称作Part，包含两个振荡器和一个滤波器。Sylenth1拥有两个Part，分别为Part A和Part B，但是同一时间用户界面只显示其中一个。Part A有一个红色的背景颜色，Part B则是黄色。

第二部分是绿色，包含一个滤波控制和一个混合器面板，它们控制从两个Part传入的声音。

第三部分是界面底部的蓝色面板，包含全部的调制选项。

最后的第四部分由一个LCD显示器形成，包含主线效果。

紧接着这四个部分，在位于界面底部的地方有一个键盘、弯音轮、调制轮和滑音控件。

3.1 音频路径

下图显示了Sylenth1内音频路径的内部结构。每条线代表立体音频信号。每个Part的两个振荡器的输出会叠加在一起，之后会进入滤波器部分。当然，将part A中振荡器的输出传入part B的滤波器也是可以的，正如虚交叉线所示那样。

滤波器的输出随后重塑于振荡包络发生器，接着再进入混合器，最终，在施加了主线效果后音频输出信号发送到宿主。

从图中可以看出，滤波控制(Filter control)面板同时控制part A和part B的滤波器。

主线效果由一个琶音器和七个串联的声音效果器组成。第一个效果器是失真，接着是移相、合唱、均衡、混响、延迟，最后是压缩。

3.2 通过Part进行工作

Part选择

你可以通过用户界面顶栏的“Part Select”按钮在两个Part之间进行切换。很容易在用户界面知道当前显示的是哪个Part，因为Part A拥有偏红背景色，而Part B则是黄色。

Solo

紧挨“Part Select”右边，你能找到名叫“Solo”的按钮，当该按钮被激活时，你将只听到当前所选Part的声音，这在你工作于复杂声音时非常有用。

3.3 控制旋钮

旋钮和控件

旋钮、滑块和其它控件可以通过单击按钮并在垂直方向上下拖拽来控制。如果你需要一个更高精度的控制，你可以在拖拽鼠标的时候按住Shift键。你也可以在旋转模式按住Alt键来控制按钮。

中心值

某些旋钮和控件具有居中的默认设置，例如平移旋钮或八度音阶以及振荡器上的音符值。双击这些控件将使它们跳转到其中心值。或者，在单击旋钮的同时按住Ctrl键可以使所有旋钮居中。

参数值显示

但凡你通过拖动旋钮更改参数值的时候，该参数值将显示在LCD屏幕上。

如果你想知道参数的确切值，但却又不想更改它，只需单击一次旋钮即可。这将在屏幕上显示其值而不更改它。

3.4 MIDI控制

Sylenth1上的大多数旋钮都可以通过MIDI Control Changes(CC)控制。每个旋钮都有一个默认的CC参数，在附录中列出。你也可以通过鼠标右键单击它们来轻松地将CC分配给旋钮。此时将弹出一个菜单，在这里可以学习或取消一个MIDI控制器。

单击“Midi Learn”并转动MIDI控制器上的旋钮。旋钮将立即链接到控制器。Sylenth1将存储它永久学习的CC参数，因此在关闭并重新启动主机后它们仍将处于活动状态。

单击“Midi Forget”以撤消学习到的值并释放控制器。

3.5 改变程序

Sylenth1 音色库(soundbanks)包含512个预置，分为四个子组(sub-banks)，每个子组有128个预置。当前选择的子组(sub-bank)和程序显示在Sylenth的LCD显示屏上。

注：这里的程序(program)指的是一个预置的音色。

有几种方法可以改变程序。

  • 通过使用宿主软件，该软件具有更改程序的功能。
  • 通过使用LCD显示屏上的程序选择箭头。
  • 通过单击LCD显示屏上的程序名称，将显示一个包含128个程序编号的弹出菜单。你可以通过单击程序名称左侧的数字切换子组(sub-banks)。
  • 通过MIDI发送一个程序更改消息(因为只有128个MIDI程序，你只能用这种方法选择前128个程序)。

3.6 复音(Polyphony)

Sylenth1是一个复音合成器，这意味着你可以同时演奏多个音符。可以通过位于用户界面顶部栏上的“Polyphony”控件来控制最大并发音符数。这个音符数可以设置在0到16之间的任何位置。

声音(voices)

声音指示器显示活动声音(voice)的数量和最大语音总数。每个振荡器可以产生几个声音。最大声音数等于所有振荡器产生的声音之和乘以复音音符的最大数量。例如，当有2个振荡器激活，每个振荡器产生3个声音时，Sylenth1将为你演奏的每个音符合成6个声音。当复音设置为4时，这意味着总共最多有24个声音。

3.7 同步

“Sync”按钮位于用户界面的顶部栏上。它强制将几个定时参数(如LFO速率和延迟时间)与宿主速度同步。

如果关闭“Sync”，则这些参数将以秒或赫兹显示其值。当它打开时，它们将在条形部分显示它们的值。一个条形包含四个节拍，所以1/4表示每个节拍一次，1/8表示每个节拍两次。Triolic和间断(punctuated)值也是可能的：“T”代表triolic，“D”代表间断音符长度。

3.8 加载、保存和编辑预置

你可以使用LCD显示屏上的“MENU”按钮加载、保存和编辑预置和音色库。

加载和保存预置和音色库

可以使用“Preset”菜单中的“Save As…”选项保存单个预置。它们以.fxp文件扩展名保存。要将预置加载到当前所选的程序编号，请从“Preset”菜单中选择“Load…”，然后选择要加载的预置文件。你可以通过按住“Shift”或“Ctrl”键一次选择多个文件。

以类似的方式，你可以使用“Bank -> Load ../ Save As..”选项加载和保存包含512个预置的音色库。 音色库以.fxb文件扩展名保存。

重命名预置

单击预置名称左侧的小点可以更改预置名称。要取消编辑，请按“Esc”键或单击编辑框外的任何位置。要应用更改，请按“Enter”键。

编辑预置

你可以复制一个预置到另一个程序编号，方法是单击“Preset”菜单中的“Copy”选项并在之后使用“Paste”或“Insert”选项。“Delete”会将一个预置从列表中删除。

如果你想要用一个默认的设置(clean setting)来创建自己的预置，可以选择“Clear”选项，该选项将初始化当前所选预置的所有参数。“Randomize”选项允许你随机化所有参数设置，如果你想要创建新的音效，这会非常有用。

使用“Reset”选项，你可以将当前所选预置的参数重置为其原始状态。这样，你可以撤消对所选预置所做的所有参数更改。

4. 各部分细述

在本章中，我们将仔细研究Sylenth1的不同部分。每个部分由几个组件组成，它们将被逐一讨论。

4.1 Part部分

如前所述，Sylenth1上有两个Part，分别为A(红色)和B(黄色)，可以使用“part select”按钮进行切换。每一个part都包含两个振荡器，一个滤波器面板和一个振幅包络(amplitude envelope)。

4.1.1 振荡器

声音(Voices)

Sylenth1合成器上的每个振荡器可以同时产生0到8个声音。你可以通过向上或向下拖动声音选择器(Voices-selector)来选择声音数量。可以通过将声音数设置为0来关闭振荡器。

波形

通过上下拖动波形选择器(Waveform selector)，你可以为所有振荡器声音选择将要生成的波形类型。这里提供8种不同的模拟波形。

音调(Pitch)

在振荡器面板的左侧，你将找到音调控制。使用“Octave”和“Note”控件，你可以将振荡器音色调到你喜欢的任何音符。使用旋钮“Fine”可以在两个半音之间对音调进行微调。双击这些控件会将它们重置为默认的0值。

有一组旋钮可用于改变振荡器的声音：

  • 音量：设置振荡器的输出音量。
  • 相位：此旋钮将更改波形的起始点。它仅在启用“Retrigger”选项时才会有作用。
  • 调整音调：在中心音调周围以一定量调整各个音色的音调。这可用于创建超级锯齿波(supersaw)悬停效果，加宽声音或添加模拟声音效果。
  • 立体声：设置各个声音之间的立体声分离级别。如果设置为0，它将产生单声道声音，如果设置为10，则声音将完全分布在左右声道上。此效果仅当声音数超过1时才有作用。
  • 声相(Pan)：将振荡器的输出声相进行左(0)或右(10)通道设置。
重新触发(Retrigger)和相位(Phase)

每次播放新音符时，“Retrigger”按钮可用于强制所有声音从波形上的完全相同位置开始。可以使用“Phase”旋钮更改该位置。如果“Phase”设置为0，则声音将在波形周期开始时重新开始。如果设置为180度，则声音将在周期的一半开始，而设置为360度会使声音从波形的末尾开始，这相当于开头。

如果只有一个振荡器处于活动状态，则无法听到相位设置，但只要有两个或多个振荡器以相同的间距运行并且两个振荡器都启用了重新触发按钮，就可以使用它创建有用的效果。例如，你可以加重bass音色的级别，或者通过调制其中一个振荡器的相位来创建类似PWM的效果。

反转(Inv)

“Inv”按钮用来反转振荡器的输出。这在创建类似PWM音效时非常有用。例如，如果添加两个锯振荡器并反转其中一个，则生成的波形将为脉冲。然后，你可以通过更改其中一个振荡器的相位来调整脉冲宽度。

复制 / 粘贴

在每个振荡器的右上角，你会找到两个小按钮分别标为“C”和“P”。使用这些按钮，你可以复制振荡器设置并将其粘贴到另一个振荡器或其他预置。

4.1.2 滤波器

输入选择(Input Select)

振荡器的输出将被滤波器重塑。通常，来自A部分的振荡器的输出将被滤波器A滤波，但是也可以将振荡器B的输出路由到滤波器A，反之亦然。这可以通过使用输入选择(Input Select)控件来完成。如果设置为“A”，仅振荡器A1和A2被滤波。如果设置为“AB”，则part A和part B的所有振荡器将被路由到滤波器。如果设置为“–”，则表示滤波器没有任何输入。输入选择控件上方的路由方案将显示内部路由。

滤波类型(Filter Type)
  • Bypass：旁通
  • Lowpass：低通
  • Bandpass：带通
  • Highpass：高通

可以使用滤波器面板右侧的开关设置衰减量。衰减可在每倍频程12dB或每倍频程24dB之间切换。

Cutoff和Resonance

可以使用“Cutoff”旋钮更改截止频率，“Resonance”旋钮设置截止频率下的滤波器谐振量。将共振设置为非常高的水平使得滤波器自振荡，这意味着生成的音调将完全由它自己合成。

驱动(Drive)

你可能已经熟悉“Cutoff”和“Resonance”旋钮，但“Drive”旋钮在合成器上并不常见。它可用于overdrive内部滤波器层次。滤波器包含四个滤波器层次，每个层次包含非线性饱和功能。驱动这些层次会增加输入信号的高次谐波，使滤波器听起来更温暖更饱满，这类似于一个模拟滤波器的工作方式。

4.1.3 振幅包络(The Amplitude Envelope)

振荡器和滤波器产生的声音可以使用幅度包络进行重塑。这里提供了常见的ADSR包络，具有“Attack”，“Decay”，“Sustain”和“Release”参数。你可以使用这些包络让声音淡入淡出，或者增加力量感等等。

请注意，将“Attack”(攻击)或“Release”(释放)设置设置为0会产生非常尖锐的开/关切换效果，这可能会导致发出咔嗒声。虽然这对于需要大量打击的声音非常有用，例如底鼓，但它可能对其他声音产生不良影响。如果是这种情况，只需给予“Attack”和“Release”一个很小的值，那么任何点击效果将被消除。

Main部分(The Main Section)

4.2.1 滤波器控制面板

滤波器控制面板包含的控件同时适用于Part A和Part B的滤波器。请注意，此面板不包含单个滤波器；它只控制A和B的滤波器。

Cutoff和Resonance

使用“Cutoff”旋钮，你可以更改滤波器A和B的截止频率。这很有用，因为它允许你使用单个旋钮控制滤波器产生的声音的音色。同样的，“Resonance”旋钮控制两个滤波器的谐振。

Keytrack

可以将滤波器的截止频率与键盘上弹奏的音符的音高相关联。这可以使用“Keytrack”旋钮完成。当此旋钮设为0时，截止频率将不依赖于播放的音符。但是，当此旋钮设置为10时，截止频率将随播放音符的音高而变化。这使你可以在播放更高或更低音符时保持(或改变)声音的音色。

如果滤波器处于自振荡状态，将“Keytrack”旋钮设置为10可以像弹奏钢琴一样调节滤波器。

暖驱动(Warm Drive)

打开“Warm Drive”按钮，使滤波器的层次(stages)采用更高的饱和度，从而引入更高的谐波。关闭此选项会使滤波器使用较少的CPU功率。如果你正在使用大量复音，那么在CPU使用率成为问题时将其关闭可能是个好主意。

4.2.2 混合器

Part A和Part B的振荡器和滤波器产生的声音使用混音器面板混合在一起。“Mix A”设定Part A的音量；“Mix B”设定Part B的音量。在施加了主线效果之后，主线音量旋钮用来设定Sylenth1音频输出的音量。

在混音器面板的右侧是一个VU表，用于测量输出电平。请注意，当红色LED点亮时通常不会出现问题，因为Sylenth1的输出没有被剪裁。如果需要，宿主软件可以降低音量来防止削波。

4.3 Modulation部分

通过提供两个ADSR包络生成器，两个低频振荡器和一组其它的源，调制面板让你能调制一整套参数。每个源都可以链接到两个不同的参数，当你单击其中一个目标显示时，它可以在出现的弹出菜单中被选择。

你可以使用目标显示旁边的小旋钮设置每个参数调制的量。将它们向右转动会产生正调制值，将它们向左转会产生负值。中间位置是默认位置，表示没有调制。如果双击旋钮，它将跳回此默认位置。

4.3.1 调制包络(The modulation envelope)

这里的调制包络是常见的ADSR包络，具有“Attack”，“Decay”，“Sustain”和“Release”设置，类似于幅度包络。它可以用于各种调制，但是常用在两个地方上，它们是滤波器截止(aka FEG)和音调(aka PEG)。

4.3.2 低频振荡器(The LFO)

低频振荡器(LFO)也是一种常见的调制源。Sylenth1上的两个LFO提供了10种不同的波形可供选择。“Rate”旋钮设置振荡器频率，“Gain”旋钮更改振幅，“Offset”旋钮可用于给LFO的输出信号添加正负常数值。如果“Offset”设置为0，则输出波将以0为中心。如果按下“Free”按钮，LFO将自行运行，并且在播放新音符时不会重新触发，也不会和宿主速度同步。

使用LFO，你可以创建各种颤音效果、清晰的声音和特殊效果。

4.3.3 Miscellaneous Modulation面板

Miscellaneous Modulation面板允许你选择其他调制源，如速度、触后、键盘、modwheel、幅度包络、调制包络和LFO。

由于这些可选输入有四个，每个输入都有两个目标，因此可为你提供多种调制选项。同样，可以使用小旋钮改变每个调制的量。

4.4 主线效果部分

用户界面中间的LCD屏幕包含七个主线音效和一个琶音器。主线效果应用于音频路径的末尾。每个效果在LCD屏幕上都有自己的设置面板。要在不同的效果面板之间切换，请单击屏幕左侧效果的名称。

单击效果名称旁边的复选框可以打开和关闭效果。如果关闭效果，它将停止处理并释放CPU时间(CPU time)。

可以使用每个效果面板右上角的“C”和“P”按钮将效果设置复制并粘贴到不同的面板上。

4.4.1 琶音器

Mode

琶音器是一个自动化的MIDI音序生成器。它以一个旋律顺序播放你在键盘上按下的音符，由“Mode”设置定义。例如，当你选择模式为“Up”时，它将按升序排列你按下的键。如果选择“Down”，它将按降序播放。“Up/Down”模式播放升序和降序的组合。“Random”将播放随机序列，“Ordered”将以你在键盘上按下它们的相同顺序来播放音符。最后是“Step Seq”和“Step Chord”模式，它们将按步进音序器定义的顺序播放音符。“Step Seq”模式一次只能播放一个音符，而在“Step Chord”模式下，你可以演奏由音序器排列的和弦。

Time

“Time”旋钮设置两个连续音符之间的持续时间，从而定义了音符的播放速度。

Gate

你可以使用“Gate”旋钮更改每个音符的长度。当“Gate”设置在一半的时候，意味着每个音符的长度是两个连续音符之间的时间的一半。

Wrap

可以使用“Wrap”参数将音符序列循环到一定数量的音符。例如，当“Wrap”设置为8时，每过8个音符排列的序列将重新开始。

Octave

“Octave”设置定义音符序列将播放多少个八度音程。

The Step Sequencer

在琶音器面板底部，你将发现一个步进音序器。它可用于按任何特定顺序排列音符。“Transpose”参数定义每个音符的音调，它被添加到你在键盘上按下key上。例如，值为+12意味着音高将恰好转换为12个音符(一个八度)。仅当琶音器“Mode”设置为“Step Seq”或“Step Chord”时，才使用“Transpose”设置。

“Velocity”参数定义每个音符的播放速度。如果Velocity设置为“-”，则根本不会播放该音符。此外，每个音符的长度可以使用“Hold”按钮进行扩展。例如，这可用于创建滑音滑动效果。

Velocity

你可以选择五种不同的方法来更改排列音符的速度。如果“Velocity”设置为“Key”，则键盘上按下音符各自所具有的不同速度都会被使用。在“Hold”模式下，所有序列的音符将具有相同的速度，这个速度是即是你最后按下的音符所保持的速度。通过将“Velocity”设置为“Step”，你也可以使用步进音序器中的速度。然后还有两种模式：“Step + Key”和“Step + Hold”。选择这些模式中的任何一个时，按下的键的速度将与步进音序器的速度混合。

Step Velocity Modulation Source

步进音序器还可以将其速度(velocity)和hold值作为输出信号用于Misc Mod面板中的调制源。例如，该信号对于创建门限效果(gate effect)很有用。为此，请选择“Step Vlcty”作为调制源并将“Mix AB”设置为目标源(destination)。然后将Mix A和B滑块设置为0，这样输出音量将完全由“Step Velocity”设置。

4.4.2 失真

Type

Sylenth1提供五种类型的失真：Overdrive、Foldback、Clip、Decimate和Bitcrush。每种类型都有不同的声音和谐波内容。你可以通过单击箭头按钮或上下拖动类型显示来切换类型。

Amount

“Amount”旋钮设置失真的粗糙度。较低的量值将产生柔和的失真效果，这可用于为声音增加温暖，就像电子管放大器一样。较高的失真量值用来产生原始的重度失真声。

Dry/Wet

“Dry / Wet”旋钮定义了失真(湿)和原始(干)信号混合的比率。值为0表示仅传递原始信号，而值为10表示仅传递失真信号。

4.4.3 Phaser

六层立体声移相器会在频谱中产生陷波(Notch)，并会使用内部的LFO上下移动陷波。

CenterFreq

“CenterFreq”旋钮设置频谱内中间陷波的位置。旋转此旋钮可使所有陷波频率升高或降低。该参数可以由内部LFO进行调制，但是你也可以使用调制面板对其进行调制，或自己打开它。这样，你可以使移相器听起来像一个多层滤波的效果。

Spread

“Spread”旋钮确定频谱中陷波之间的距离。一个底的spread值将陷波推到一起，并产生经典的移相器声音。较高的spread值可用于创建类似滤波的效果。

LR Offset

使用“LR Offset”旋钮，你可以在左右声道的中心频率之间更改频率偏移量。这将产生立体声相位效果。

4.5 键盘部分

用户界面底部的键盘部分由一个5倍音程的键盘、一个弯音轮、一个调制轮和一个滑音控件组成。

弯音轮

最左边的轮子是弯音轮。它可以通过MIDI或鼠标控制，它会向上或向下弯曲音高。你可以使用“BendRange”控件在半音符中指定弯曲范围。

调制轮

调制轮位于弯音轮旁边。它可以用作Miscellaneous Modulation面板中的调制源，用于任何类型的调制。

Mono Legato

当“Mono Legato”按钮被激活时，Sylenth1将被强制进入单声道模式，并且顶部栏上的复音(polyphony)设置将被丢弃。每当按下另一个键时，它也将停止重新触发音符。这使你可以在音高和速度上上下滑动音符，而无需重新启动新音符。

滑音(Portamento)

“Portamento”旋钮设置从一个音符滑动到另一个音符所需的时间。较小的值会使音符快速滑动，而较高的值会使音符滑动较慢。

有两种滑音模式，Normal(N)和Slide(S)，可以使用“Portamento”旋钮旁边的“Mode”开关进行选择。 在“Slide”模式下，它总是将音高滑动到播放的下一个音符。但是，在“Normal”模式下，只有按下至少一个其它键时才会滑动音符，但是一次只演奏一个音符时它不会滑动。

5. 减少CPU使用率的小建议

当同时使用大量软件合成器和音频效果时(通常在制作音乐时会这样做)，CPU使用率可能会成为一个问题。 以下是将Sylenth1的CPU使用率降至最低的一些提示。

复音

关于CPU使用率最重要的设置可能是复音。对于每个复音音符，几乎会拷贝一份完整的音频路径副本(主线效果除外)。 因此，同时播放两个音符的成本几乎是一个音符所需CPU的两倍。始终都要尽量将复音保持在最低限度。

振荡器

你使用的振荡器越多，所需的CPU就要越强。此外，4个带有1个语音的振荡器比带有4个语音的1个振荡器需要更多的CPU。所以，尝试使用数量更少的带有较多声音的振荡器是一个好主意。

包络生成器

尽量保持“Decay”和“Release”参数尽可能小。释放时间较短的声音通常也会使用较少的复音音符。

滤波器

如果你不使用滤波器，请将它们切换到旁路模式。这将迫使他们停止处理。如果使用大量复音，请关闭“Warm Drive”功能以节省一些额外的CPU时间。

主线效果

关闭你不使用或不需要的任何效果。如果你运行多个Sylenth1实例，所有实例都具有相同类型的混响，你可能最好使用一个外部混响发送效果，并将所有音频信号路由到它。请注意，尽管如此一些混响VST比Sylenth1的混响消耗更多的CPU时间！

音调和相位调制

调制振荡器的音调和相位对于具有大量voices的声音非常苛刻，因为每个voice都需要被调制。如果你需要调制音调或相位，请尝试限制所使用的声音总数。

如果你正在寻找颤音效果，你也可以使用合唱而不是音调调制来完成此操作。将合唱“Dry/Wet”设置为10，“Delay”尽可能小，并确保关闭双模式(Dual Mode)。这是一个对CPU更加友好的解决方案，尤其是在播放大量声音(voice)时。
