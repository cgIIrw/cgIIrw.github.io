Sylenth1手册简明翻译

发表于 2019-08-29 | 分类于 合成器 | 阅读次数:
字数统计: 1,940 | 阅读时长 ≈ 7

本文是Sylenth1Manual的简明翻译。

概览

Sylenth1的用户界面分成四个部分，每个部分拥有不同的颜色以便彼此区分。第一个部分被称作Part，包含两个振荡器和一个滤波器。Sylenth1拥有两个Part，分别为Part A和Part B，但是同一时间用户界面只显示其中一个。Part A有一个红色的背景颜色，Part B则是黄色。

第二部分是绿色，包含一个滤波控制和一个混合器面板，它们控制从两个Part传入的声音。第三部分是界面底部的蓝色面板，包含全部的调制选项。最后的第四部分由一个LCD显示器形成，包含主线效果。

紧接着这四个部分，在位于界面底部的地方有一个键盘、弯音轮、调制轮和滑音控制。

音频路径

下图显示了Sylenth1内音频路径的内部结构。每条线代表立体音频信号。每个Part的两个振荡器的输出会叠加在一起，之后会进入滤波器部分。当然，将part A中振荡器的输出传入part B的滤波器也是可以的，正如虚交叉线所示那样。

过滤器的输出随后重塑于振荡包络发生器，接着再进入混合器，最终，在施加了主线效果后音频输出信号发送到宿主。

从图中可以看出，滤波控制面板同时控制part A和part B的滤波器。

主线效果由一个琶音器和七个串联的声音效果器组成。第一个效果器是失真，接着是移相、合唱、均衡、混响、延迟，最后是压缩。

通过Part进行工作

Part选择

你可以通过用户界面顶栏的“Part Select”按钮在两个Part之间进行切换。很容易在用户界面知道当前显示的是哪个Part，因为Part A拥有偏红背景色，而Part B则是黄色。

Solo

紧挨“Part Select”右边，你能找到名叫“Solo”的按钮，当该按钮被激活时，你将只听到当前所选Part的声音，这在你工作于复杂声音时非常有用。

控制旋钮

旋钮和控制

旋钮、滑块和其它控件可以通过单击按钮并在垂直方向上下拖拽来控制。如果你需要一个更高精度的控制，你可以在拖拽鼠标的时候按住Shift键。你也可以在旋转模式按住Alt键来控制按钮。

中心值

某些旋钮和控件具有居中的默认设置，例如平移旋钮或八度音阶以及振荡器上的音符值。双击这些控件将使它们跳转到其中心值。或者，在单击旋钮的同时按住Ctrl键可以使所有旋钮居中。

参数值显示

但凡你通过拖动旋钮更改参数值的时候，该参数值将显示在LCD屏幕上。

如果你想知道参数的确切值，但却又不想更改它，只需单击一次旋钮即可。这将在屏幕上显示其值而不更改它。

MIDI控制

Sylenth1上的大多数旋钮都可以通过MIDI Control Changes(CC)控制。每个旋钮都有一个默认的CC参数，在附录中列出。你也可以通过鼠标右键单击它们来轻松地将CC分配给旋钮。此时将弹出一个菜单，在这里可以学习或取消一个MIDI控制器。

单击Midi Learn并转动MIDI控制器上的旋钮。旋钮将立即链接到控制器。Sylenth1将存储它永久学习的CC参数，因此在关闭并重新启动主机后它们仍将处于活动状态。

单击Midi Forget以撤消学习到的值并释放控制器。

改变程序

Sylenth1 soundbanks包含512个预设，分为四个子组(sub-banks)，每个子组有128个预设。当前选择的子库(sub-bank)和程序显示在Sylenth的LCD显示屏上。

有几种方法可以改变程序。

  • 通过使用宿主软件，该软件具有更改程序的功能。
  • 通过使用LCD显示屏上的程序选择箭头。
  • 通过单击LCD显示屏上的程序名称，将显示一个包含128个程序编号的弹出菜单。你可以通过单击程序名称左侧的数字切换子库(sub-banks)。
  • 通过MIDI发送一个程序更改消息(因为只有128个MIDI程序，你只能用这种方法选择前128个程序)。

复音

Sylenth1是一个复音合成器，这意味着你可以同时演奏多个音符。可以通过位于用户界面顶部栏上的Polyphony控件来控制最大并发音符数。这个音符数可以设置在0到16之间的任何位置。

声音

声音指示器显示活动语音的数量和最大语音总数。每个振荡器可以产生几个声音。最大声音数等于所有振荡器产生的声音之和乘以复音音符的最大数量。例如，当有2个振荡器激活，每个振荡器产生3个声音时，Sylenth1将为你演奏的每个音符合成6个声音。当复音设置为4时，这意味着总共最多有24个声音。

同步

“同步”按钮位于用户界面的顶部栏上。它强制将几个定时参数(如LFO速率和延迟时间)与宿主速度同步。

如果关闭“同步”，则这些参数将以秒或赫兹显示其值。当它打开时，它们将在条形部分显示它们的值。一个条形包含四个节拍，因此值1/4表示每个节拍一次，值1/8表示每个节拍两次。Triolic和punctuated值也是可能的：T代表triolic，D代表间断音符长度。

加载、保存和编辑预置

您可以使用LCD显示屏上的MENU按钮加载，保存和编辑预置和音色库。

加载和保存预置和音色库

可以使用“Preset”菜单中的“Save As…”选项保存单个预置。它们以.fxp文件扩展名保存。要将预置加载到当前所选的程序编号，请从“Preset”菜单中选择“Load…”，然后选择要加载的预设文件。您可以通过按住Shift或Ctrl键一次选择多个文件。

以类似的方式，您可以使用“Bank -> Load ../ Save As..”选项加载和保存包含512个预置的音色库。 音色库以.fxb文件扩展名保存。

重命名预置

单击预置名称左侧的小点可以更改预置名称。要取消编辑，请按Esc键或单击编辑框外的任何位置。要应用更改，请按Enter键。

编辑预置

你可以通过单击“预置”菜单中的“复制”选项并在之后使用其他程序编号中的“粘贴”或“插入”选项将预设复制到另一个程序编号。删除将从列表中删除预置。

如果要创建自己的预置并以干净设置开始，可以选择“清除”选项，该选项将初始化当前所选预置的所有参数。 Randomize选项允许您随机化所有参数设置，如果您想要创建新的音效，这些设置非常有用。

使用“重置”选项，您可以将当前所选预置的参数重置为其原始状态。这样，您可以撤消对所选预置所做的所有参数更改。
